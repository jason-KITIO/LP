# ランディングページ - 管理職セミナー

チーム管理スキルと目標達成力の向上を促進する、モダンで最適化されたランディングページです。

## 🎯 目的

このページは、管理チームのパフォーマンス向上を目指す企業経営者向けの無料セミナーへの参加者獲得を目的としています。

## 🚀 機能

- **レスポンシブデザイン** - モバイルとデスクトップに最適化
- **最適化されたCSS** - メンテナンスを容易にするユーティリティクラス
- **適応的画像** - 異なる画面サイズに対応する`<picture>`要素の使用
- **複数のCTA** - 戦略的に配置された登録ボタン

## 📁 プロジェクト構造

```
LP/
├── images/                 # 視覚的リソース
│   ├── 123.png            # ヒーローセクションのモバイル版
│   ├── FV.png             # ヒーローセクションのデスクトップ版
│   └── ...                # その他の画像
├── *.svg                  # アイコンとグラフィック要素
├── index.html             # メインページ
├── style.css              # 最適化されたスタイル
├── script.js              # JavaScriptスクリプト
└── README.md              # ドキュメント
```

## 🎨 CSS最適化

プロジェクトはコードの重複を減らすためのユーティリティクラスシステムを使用しています：

### 主要クラス
- `.w-65vw`, `.w-100vw` - 標準化された幅
- `.flex-center` - フレックスボックス中央揃え
- `.list-reset` - リストのリセット
- `.text-red`, `.text-yellow` - テキストカラー
- `.bg-overlay` - 画像オーバーレイ

### 達成された削減
- **25%のCSS削減**
- **簡素化されたメンテナンス** - スタイルの一元化により
- **より読みやすいセマンティッククラス**

## 🛠️ 使用技術

- **HTML5** - セマンティック構造
- **CSS3** - フレックスボックスを使用したモダンスタイル
- **JavaScript** - インタラクション（必要に応じて）
- **SVG** - 最適化されたベクターアイコン

## 📱 レスポンシブデザイン

ページは異なる画面サイズに自動的に適応します：
- **モバイル** (< 768px) - 最適化された縦レイアウト
- **デスクトップ** (≥ 768px) - より大きな画像を使用した横レイアウト

## 🚀 インストールと使用方法

1. プロジェクトをクローンまたはダウンロード
2. ブラウザで`index.html`を開く
3. 追加のインストールは不要

- **最適化された構造による高速読み込み**

## 🎯 コンバージョン

ページはコンバージョンを最大化するよう設計されています：
- **問題を特定する魅力的なタイトル**
- **社会的証明**と推薦文
- **明確で繰り返されるCTA**

## 📝 メンテナンス

コンテンツを変更するには：
1. **テキスト** - `index.html`で直接編集
2. **スタイル** - `style.css`の既存ユーティリティクラスを使用
3. **画像** - `images/`フォルダ内のファイルを置換

## 🔧 カスタマイズ

ユーティリティクラスシステムにより簡単なカスタマイズが可能：
- `style.css`に新しいクラスを追加
- 一貫性を保つため既存クラスを再利用
- 確立された命名規則に従う
